## **2. Cursor Navigation**  
- **`CTRL + A`** â†’ Pindah ke awal baris.  
- **`CTRL + E`** â†’ Pindah ke akhir baris.  
- **`CTRL + â† / â†’`** â†’ Pindah ke awal kata sebelumnya/berikutnya.

## **3. Deleting Text**  
- **`CTRL + U`** â†’ Hapus teks dari kursor ke awal baris.  
- **`CTRL + K`** â†’ Hapus teks dari kursor ke akhir baris.  
- **`CTRL + W`** â†’ Hapus kata sebelum kursor.  

## **4. Undo & Paste**  
- **`CTRL + Y`** â†’ Tempel teks yang sebelumnya dihapus.  

## **5. Process Management**  
- **`CTRL + C`** â†’ Menghentikan proses yang sedang berjalan (mengirimkan sinyal `SIGINT`).  
- **`CTRL + Z`** â†’ Menangguhkan proses ke latar belakang (mengirimkan sinyal `SIGTSTP`).  

## **6. Terminal Management**  
- **`CTRL + D`** â†’ Menutup sesi terminal atau menandakan EOF (End of File).  
- **`CTRL + L`** â†’ Membersihkan layar terminal (alternatif dari perintah `clear`).  
- **`CTRL + S`** â†’ Menghentikan output ke terminal (berguna jika output terlalu panjang).  
- **`CTRL + Q`** â†’ Melanjutkan output yang dihentikan dengan `CTRL + S`.  
- **`CTRL + Shift + C`** â†’ Menyalin teks yang dipilih di terminal.  
- **`CTRL + Shift + V`** â†’ Menempelkan teks di terminal.  

## **7. Command History**  
- **`CTRL + R`** â†’ Mencari perintah sebelumnya dalam history.  
- **`â†‘ / â†“`** â†’ Menelusuri riwayat perintah yang telah digunakan.  
- **`CTRL + J`** â†’ Menjalankan perintah yang ada di baris history tanpa harus menekan Enter.  

## **8. Application & Window Management**  
- **`ALT + TAB`** â†’ Berpindah antar aplikasi yang terbuka.  
- **`ALT + F4`** â†’ Menutup jendela aktif.  
- **`ALT + F5`** â†’ Mengembalikan ukuran jendela ke ukuran sebelumnya.  
- **`ALT + F7`** â†’ Memindahkan jendela (tekan dan tahan tombol ALT, lalu gunakan mouse atau tombol panah).  
- **`ALT + F8`** â†’ Mengubah ukuran jendela (tekan dan tahan tombol ALT, lalu gunakan mouse atau tombol panah).  
- **`ALT + F9`** â†’ Meminimalkan jendela.  
- **`ALT + F10`** â†’ Memaksimalkan jendela.  

## **9. Switching TTY (Virtual Terminal)**  
- **`CTRL + ALT + F1`** â†’ Berpindah ke TTY1.  
- **`CTRL + ALT + F2`** â†’ Berpindah ke TTY2.  
- **`CTRL + ALT + F3`** â†’ Berpindah ke TTY3.

- **`CTRL + ALT + F5`** â†’ Berpindah ke TTY5.  
- **`CTRL + ALT + F6`** â†’ Berpindah ke TTY6.  
- **`CTRL + ALT + F7`** â†’ Kembali ke lingkungan grafis (biasanya).  

## **10. Other Shortcuts**  
- **`CTRL + Space`** â†’ Memunculkan daftar saran atau penyelesaian perintah.  
- **`CTRL + G`** â†’ Membatalkan perintah yang sedang berjalan atau pencarian history.  
- **`CTRL + H`** â†’ Menampilkan bantuan untuk perintah.  

## **11. Zoom**  
- **`CTRL + +`** â†’ Perbesar tampilan.  
- **`CTRL + -`** â†’ Perkecil tampilan.  

## **Tips Tambahan**  
- Sebagian besar shortcut di atas berlaku untuk terminal berbasis teks (TTY) dan emulator terminal di lingkungan grafis.  
- Beberapa shortcut mungkin berbeda tergantung pada distribusi Linux dan desktop environment yang Anda gunakan.  
- Anda dapat menyesuaikan beberapa shortcut melalui pengaturan terminal atau desktop environment Anda.
